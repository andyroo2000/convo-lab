# Kanjium Pitch Accent Data

This directory includes Kanjium Japanese pitch accent data from:

https://github.com/mifunetoshiro/kanjium

The upstream project states that the accent data is distributed under the
Creative Commons Attribution-ShareAlike 4.0 International License.

ConvoLab uses `accents.txt` as a lookup table for Japanese pitch accent
numbers. The implementation stores derived structured pitch accent metadata on
study cards and renders diagrams client-side.
